# DVLD (Driving & Vehicle License Department System)

This project is a real-world simulation of a Driving License Management System designed to handle the complete lifecycle of license issuance.

## ✨ Key Features:
- Multi-step application workflow (Vision, Theory, Practical tests)
- License issuance, renewal, replacement, and international license support
- Advanced validation rules and business logic enforcement
- Modular and reusable architecture

## 🏗 Architecture:
- Clean Architecture (UI / Services / Data Access)
- Repository & Unit of Work patterns
- Service Layer with business workflows
- Event-driven WinForms UI

## 🧰 Built With:
- C# / .NET Framework (WinForms)
- SQL Server
- Custom DevToolKit (internal reusable framework)

## 🎯 Purpose:
This project demonstrates strong software engineering practices, system design thinking, and the ability to build scalable desktop applications with real-world complexity.
